# Task Tracker CLI

A simple Command Line Interface (CLI) application to manage your daily tasks. This tool allows you to add, update, delete, and track the status of your tasks directly from your terminal.
https://roadmap.sh/projects/task-tracker

## Features

* **Add Tasks:** Quickly create new tasks with a description.
* **Update/Delete:** Modify existing tasks or remove them when they are no longer needed.
* **Status Management:** Mark tasks as `todo`, `in-progress`, or `done`.
* **Filtering:** List all tasks or filter them by their current status.
* **Data Persistence:** Tasks are stored in a local JSON file.
